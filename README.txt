FILE FORMATS

[Countries.txt]
  - all individual values are line separated
  - first line contains number of countries that it contains 
  - next four lines specifies indexes of the upcoming countries,
    indexing starting from one
  - rest of the lines suppose to create list of countries in 
    alphabetical order that is at least as long as big is number 
    on the first line of the file
